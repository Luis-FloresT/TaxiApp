# TaxiWhatsApp Frontend

Panel de operadores para TaxiWhatsApp.

## Desarrollo

```bash
cp .env.example .env
npm install
npm run dev
```

Por defecto se conecta al backend en:

```text
http://localhost:3000
```

Puedes cambiarlo con:

```bash
VITE_API_URL=http://localhost:3000
```

## Scripts

```bash
npm run lint
npm run build
npm run preview
```
